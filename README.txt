Checkers Game Instructions
==========================

The game provides two modes:

1. 2-Player (Hotseat)
   - Default mode.
   - Players take turns on the same computer.
   - White moves first.

2. Play vs CPU
   - Click the "Play vs CPU" button to toggle AI.
   - The AI plays as Black.
   - When active, the computer will automatically make moves for Black.

To Play:
1. Ensure the server is running (`python manage.py runserver`).
2. Open http://127.0.0.1:8000/ in a web browser.
